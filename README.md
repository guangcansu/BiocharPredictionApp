# BiocharPredictionApp
This ML GUI predicts biochar properties (yield, HHV, elements) from lignocellulosic biomass dry torrefaction. Evaluating six models (GBM, RF, SVR, ANN, DNN, MARS) with 11 inputs, GBM proved most accurate. The optimized GBM algorithm is built into a Tkinter app, enabling researchers to easily predict biochar outputs.
